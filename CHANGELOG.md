### Added
- Next.js 16 project scaffold (TypeScript, Tailwind CSS v4, App Router)
- Prisma schema with all core models (User, Household, Profile, Transaction, Category, RecurringRule, Goal, PaymentConfirmation, TempFile)
- Docker Compose setup with optional bundled PostgreSQL 16 — plug in your own database by setting `DATABASE_URL`
- shadcn/ui new-york style component primitives
- Vitest test setup
- `.env.example` with all accepted configuration values documented
